# MyWallet PWA

A lightweight Progressive Web App to store all your membership cards, loyalty cards, shopping cards, coupons, and more — entirely offline in your browser.

## Features

- 📷 **Add cards** by uploading any image or screenshot
- 👆 **Swipe left/right** to navigate between cards
- 🏠 **Home screen** with a visual grid of all your cards for quick access
- ✏️ **Editable titles** — tap the card name to rename it
- 🗑️ **Delete cards** individually
- 💾 **Offline storage** — everything saved locally in your browser (no server, no account)
- 📱 **Install as PWA** — add to your home screen on iOS or Android

## How to host on GitHub Pages

1. Create a new GitHub repository (e.g. `mywallet`)
2. Upload `index.html` to the repo root
3. Go to **Settings → Pages**
4. Set source to **Deploy from branch → main → / (root)**
5. Your app will be live at `https://yourusername.github.io/mywallet/`

## Usage

- Tap **＋** to add a new card
- Tap any card on the home screen to open it
- Swipe **left/right** to flip through cards
- Tap the card **title** or **✎ pencil icon** to rename
- Tap **🗑** in the top right to delete the current card
- Tap **← Home** to go back to the grid

## Privacy

All data is stored **only on your device** using `localStorage`. Nothing is sent to any server.

## Install on phone

- **iOS (Safari):** tap the Share button → "Add to Home Screen"
- **Android (Chrome):** tap the ⋮ menu → "Add to Home Screen" or "Install App"
